# Customer Churn Prediction (Machine Learning)

## Overview
This project uses machine learning to predict whether a customer will churn based on tenure and monthly charges.

## Tools Used
- Python
- Pandas
- scikit-learn
- Google Colab

## Model
- Logistic Regression
- Train/Test split
- Accuracy evaluation

## Purpose
This project demonstrates foundational machine learning skills including data cleaning, model training, and business interpretation.
